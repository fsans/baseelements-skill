# Time — BaseElements Plugin

## BE_TimeCurrentMilliseconds

```
BE_TimeCurrentMilliseconds
```

**Description**

Returns the current time as number of milliseconds. Effectively the same as GetAsNumber ( Get ( CurrentTimestamp ) ).

**Parameters**

None

**Version History**

* 2.0.0 : First Release

**Compatibility**

| Platform | Compatibility |
|-----------|-----------|
| Status | Active |  
| Mac FMP | Yes  |  
| Win FMP | Yes  |  
| FMS | Yes  |  
| iOS | Yes  |  
| Linux | Yes  |  

---

## BE_TimeUTCMilliseconds

```
BE_TimeUTCMilliseconds
```

**Description**

Returns UTC time as number of milliseconds - the same as the time from BE_TimeCurrentMilliseconds but in the UTC timezone.

**Parameters**

None

**Version History**

* 2.0.0 : First Release
* 4.0.2 : Renamed from BE_UTCMilliseconds
